FitFlow is a modern and interactive fitness web application designed to help users manage their health and fitness journey. It allows users to create a profile, calculate BMI, track workouts, and manage diet plans — all in one place.
🚀 Features
👤 User Profile Management
📊 BMI Calculator
🏋️ Workout Planning
🥗 Diet Tracking
🔥 Daily Calorie Tracking
🌙 Dark Mode UI
📱 Responsive Design (Mobile Friendly)
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
💡 How It Works
User enters personal details (name, age, height, weight)
Selects fitness goal:
Weight Loss
Maintain
Muscle Gain
App calculates:
BMI
Daily targets
Displays fitness insights and plan
