💪 FitFlow – Fitness Web App

FitFlow is a modern and interactive fitness web application designed to help users manage their health and fitness journey efficiently.

---

🚀 Features

- 👤 User Profile Management
- 🧮 BMI Calculator
- 🏋️ Workout Planning
- 📊 Diet Tracking
- 🔥 Daily Calorie Tracking
- 🎨 Responsive and Clean UI Design

---

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

⚙️ How It Works

1. User enters personal details (age, height, weight)
2. Selects fitness goal (Weight Loss / Muscle Gain / Maintain)
3. App calculates BMI and targets
4. Displays fitness insights and plans
